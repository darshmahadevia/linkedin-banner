# LinkedIn Banner Studio

Generate LinkedIn banners with curated presets, live editing, and one-click PNG export. Built with Next.js, Tailwind, and shadcn/ui.

## Features

- Preset-driven designs with multiple layout styles
- Live editing for name, headline, company, location, website, email, phone, and tagline
- Toggle visibility for every banner element
- PNG export at the correct 1584 x 396 LinkedIn banner size

## Getting Started

```bash
npm install
npm run dev
```

Open http://localhost:3000

## Usage

1. Pick a preset in the left panel.
2. Edit your details and toggle elements on/off.
3. Click "Download PNG" to export a banner.

## Tech Stack

- Next.js (App Router)
- Tailwind CSS
- shadcn/ui
- html-to-image

## Notes

- Keep important text away from the left edge (LinkedIn overlays the profile photo).
